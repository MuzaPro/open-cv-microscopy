Welcome to Spheroid Analyzer

Spheroid Analyzer is a Python script designed to automate the analysis of microscope images of spheroids (3D cell cultures). The script identifies spheroids in images, measures their size, shape, and fluorescence intensity.

What This Documentation Covers

This guide provides comprehensive information about the Spheroid Analyzer script, including how to set it up, how it works under the hood, and how to troubleshoot common issues.

Key Features

Automated Detection

Automatically finds spheroids in microscope images using advanced edge detection.

Precise Measurements

Measures size, shape, and fluorescence intensity with accurate pixel-to-micron conversion.

Batch Processing

Analyzes entire experiments with multiple conditions and timepoints in a single run.

Getting Started

Prerequisites

Before using Spheroid Analyzer, make sure you have:

  • Python 3.6 or newer installed
  • Required libraries: OpenCV, NumPy, Pandas, Matplotlib, SciPy
  • Organized microscope images (see folder structure below)

Configuration

The script has several key parameters you need to configure at the top of the file:

Parameter Description
main_folder_path Path to the main folder containing experiment subfolders
output_file_path Where to save the Excel results (must end with .xlsx)
one_contour Set to True to detect only one spheroid per image, False for multiple
show_images Set to True to display images during processing
min_size Minimum size threshold (in pixels squared)
max_size Maximum size threshold (in pixels squared)
pixel_to_micron Conversion factor from pixels to micrometers (depends on microscope)
identifier Text in filenames that identifies phase contrast images (e.g., "c1")
t_test Set to True to perform statistical analysis between conditions

Important Note

The pixel_to_micron value is critical for accurate measurements. Consult your microscope's documentation to find the correct value for your magnification settings.

Running the Script

  1. Configure the parameters at the top of the script
  2. Make sure your images are organized correctly
  3. Run the script using Python: python spheroid_analyzer.py
  4. If show_images = True, you'll see processing visualizations
  5. Results will be saved to the specified Excel file

How It Works

Analysis Workflow

Spheroid Analyzer follows a systematic pipeline to detect, measure, and analyze spheroids in microscope images.

Analysis Pipeline Overview

Key Steps in Analysis Pipeline:

  1. Input Processing: Find folders and images
  2. Image Pre-processing: Apply Sobel filter, blur, create mask
  3. Contour Detection: Find and filter potential spheroids
  4. Measurement: Calculate size, shape, fluorescence intensity
  5. Results Storage: Save to Excel and generate visualizations

Filtering Criteria:

  • Size Check: Area between min_size and max_size
  • Circularity Check: Circularity > 0.5
  • Solidity Check: Solidity between 0.4 and 1.8
  • Aspect Ratio Check: Ratio between 0.25 and 4

Analysis Steps in Detail

1. Folder and File Processing

  • The script begins by identifying all condition folders within the main folder
  • Folders are sorted by the time suffix in their names
  • Within each folder, all images containing the identifier (e.g., "c1") are found

2. Image Pre-processing

  • Sobel Filter: Applied to enhance edges in the image
  • Gaussian Blur: Reduces noise in the image
  • Thresholding: Converts the image to black and white using adaptive methods
  • Morphological Operations: Cleans up the binary image by filling holes and smoothing edges

3. Contour Detection

  • The script finds all outlines (contours) in the processed image
  • Each contour is a potential spheroid
  • A series of filters is applied to determine which contours are valid spheroids

4. Contour Filtering

A contour must pass all these tests to be considered a valid spheroid:

  • Size Test: Area must be between min_size and max_size
  • Circularity Test: Must have a circularity value > 0.5
  • Solidity Test: Must have a solidity between 0.4 and 1.8
  • Aspect Ratio Test: Width-to-height ratio must be between 0.25 and 4

5. Measurements

For each valid spheroid, the script measures:

  • Area: Size in both pixels and square micrometers
  • Aspect Ratio: Ratio of width to height
  • Solidity: How "solid" the spheroid appears
  • Fluorescence Intensity: Average brightness in fluorescence channels

6. Results Storage

  • All measurements are organized by condition and timepoint
  • Results are saved to an Excel file with one sheet per condition
  • Optional statistical analysis (t-tests) can be performed to compare conditions

Visualization

When show_images = True, the script displays images at various stages of processing:

  • Sobel filtered image
  • Binary mask
  • Processed mask after morphological operations
  • Final result with detected contours highlighted

Key Insight

The most critical parts of the workflow are the contour filtering steps. If the script isn't detecting spheroids correctly, adjusting the filtering criteria often solves the problem.

Technical Components

Libraries and Dependencies

The script relies on several specialized Python libraries to perform its analysis tasks.

Core Libraries

OpenCV (cv2)

What it is: A powerful image processing toolkit.

What it does: Handles image reading, filtering, contour detection, and measurements.

In simple terms: The "eyes" that see and analyze the images.

OpenCV functions used include:

  • imread: Reads image files from disk
  • findContours: Finds outlines of objects in binary images
  • contourArea: Measures the area of a contour
  • drawContours: Draws contours on images for visualization
  • GaussianBlur: Reduces noise in images
  • Sobel: Enhances edges in images

NumPy (numpy)

What it is: A library for numerical computations.

What it does: Stores and manipulates image data as arrays of numbers.

In simple terms: The "calculator" that handles the math behind image processing.

NumPy functions used include:

  • zeros_like: Creates arrays filled with zeros
  • uint8: Defines 8-bit unsigned integer data type
  • pi: Mathematical constant used in circularity calculations

Pandas (pd)

What it is: A data analysis and manipulation tool.

What it does: Organizes and stores measurements in tables.

In simple terms: The "record keeper" that organizes all the results.

Pandas functions used include:

  • DataFrame: Creates tables to store measurements
  • concat: Combines multiple sets of results
  • to_excel: Saves results to Excel files

Matplotlib (plt)

What it is: A plotting and visualization library.

What it does: Creates visual output showing the detected spheroids.

In simple terms: The "artist" that draws pictures to help you see what's happening.

Matplotlib functions used include:

  • figure: Creates a new figure for visualization
  • subplot: Organizes multiple images in a grid
  • imshow: Displays images
  • savefig: Saves figures to disk

SciPy (stats)

What it is: A scientific computing toolkit.

What it does: Performs statistical tests to compare experimental conditions.

In simple terms: The "statistician" that tells you if your results are significant.

SciPy functions used include:

  • ttest_ind: Performs t-tests between different conditions

Image Analysis Concepts

Sobel Filter

A mathematical operation that calculates the gradient of pixel intensities, highlighting areas where intensity changes rapidly (edges).

Adaptive Thresholding

A technique that converts a grayscale image to binary (black and white) by comparing each pixel to the average of its neighbors, rather than using a single threshold value for the entire image.

Morphological Operations

Mathematical operations that modify the shape of objects in binary images:

  • Closing: Fills small holes and connects nearby objects
  • Opening: Removes small noise and separates objects
  • Dilation: Expands objects

Contour Properties

  • Area: Number of pixels within the contour
  • Perimeter: Length of the contour boundary
  • Circularity: How close the shape is to a perfect circle (4œÄ √ó Area √∑ Perimeter¬≤)
  • Convex Hull: The smallest convex shape that contains the contour
  • Solidity: Ratio of contour area to its convex hull area

Troubleshooting

Common Issues and Solutions

If the script isn't working as expected, check these common issues and their solutions.

No Spheroids Detected

Symptoms

The script runs but doesn't find any valid spheroids, even though they're clearly visible in the images.

Possible Causes and Solutions:

  1. Size thresholds are incorrect
    • Solution: Adjust min_size and max_size parameters
    • Tip: Turn on show_images to see debug information about contour sizes
  2. Identifier doesn't match your filenames
    • Solution: Check your image filenames and update the identifier parameter
  3. Image contrast is too low
    • Solution: Pre-process your images to enhance contrast, or adjust the code to use stronger edge enhancement
  4. Filtering criteria are too strict
    • Solution: Modify the circularity, solidity, or aspect ratio thresholds in the process_contours function

Wrong Spheroids Detected

Symptoms

The script detects objects, but they're not the spheroids you're interested in, or it's detecting artifacts.

Possible Causes and Solutions:

  1. Size thresholds are too permissive
    • Solution: Narrow the range between min_size and max_size
  2. Debris or artifacts in images
    • Solution: Increase the min_size parameter or make the circularity threshold higher
  3. Multiple spheroids in each image
    • Solution: Set one_contour = False if you want to detect all spheroids
    • Or keep one_contour = True if you only want the largest one

Measurements Seem Incorrect

Symptoms

The script detects spheroids but the measurements don't match what you expect.

Possible Causes and Solutions:

  1. Incorrect pixel-to-micron conversion
    • Solution: Check your microscope's documentation and update the pixel_to_micron parameter
    • Tip: Calibrate using an object of known size
  2. Contour detection is including too much or too little area
    • Solution: Adjust the thresholding and morphological operations in the process_contours function

Script Crashes or Shows Errors

Symptoms

The script terminates with an error message instead of completing.

Possible Causes and Solutions:

  1. Missing libraries
    • Solution: Install required libraries using pip: pip install opencv-python numpy pandas matplotlib scipy
  2. File path errors
    • Solution: Check that all paths are correct and use raw strings (prefix with r) for Windows paths
  3. Unsupported image formats
    • Solution: Ensure your images are in a format OpenCV can read (.tif, .jpg, .png)
  4. Permission errors when saving results
    • Solution: Check that you have write permission to the output directory

Adapting to a New Microscope

When switching to a new microscope, you'll typically need to adjust:

  1. pixel_to_micron value: Get this from the new microscope's specifications
  2. Channel identifiers: Update the identifier parameter and channel references in the code
  3. Size thresholds: May need adjustment if magnification has changed
  4. Image preprocessing: Different microscopes produce images with different characteristics

Expert Tip

When adapting to a new microscope, start by processing a few sample images with show_images = True. This will help you understand how the new images are being processed and what adjustments might be needed.
